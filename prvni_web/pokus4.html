Stránka s Bootstrap 5.3

Tato stránka využívá CSS framework **Bootstrap 5.3**. Všimněte si, že menu nahoře je **plně funkční responzivní Navbar**.

Toto je ukázka Bootstrap komponenty (Alert s třídou `alert-info`).

Systém mřížky (`row` a `col-md-6`) zajišťuje, že se tento text na středních a větších obrazovkách zobrazí vedle sebe.

Zmenšete okno prohlížeče, abyste viděli, jak se menu sbalí do ikony (hamburgeru) a jak se tato sekce zarovná pod sebe.
